# Number Checker

A simple JavaScript utility that checks and describes the properties of a number.

## How to Run

### Option 1: VSCode (Run Without Debugging)

1. Open the file `numberChecker.js` in VSCode
2. Press `Ctrl + F5`
3. Output will appear in the **Debug Console**

### Option 2: Terminal (Node.js)

1. Make sure **Node.js** is installed:
   ```
   node -v
   ```

2. Open the VSCode Terminal:
   - Press `Ctrl + ` `` ` `` (backtick)
   - Or go to **Terminal → New Terminal**

3. Run the file:
   ```
   node numberChecker.js
   ```

4. Output will appear in the **Terminal**

## Expected Output

```
positive: true, negative: false, zero: false, even: true, odd: false
positive: false, negative: true, zero: false, even: false, odd: true
positive: true, negative: false, zero: true, even: true, odd: false
positive: false, negative: true, zero: false, even: false, odd: true
```
